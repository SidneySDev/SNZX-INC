Instalação Manual no Firefox

Os usuários precisarão instalar a extensão manualmente no Firefox. Para isso, eles devem seguir os seguintes passos:

Abra o Firefox.

Vá para o menu (três linhas horizontais no canto superior direito) e selecione "Add-ons and themes" (ou "Complementos e temas").

No canto superior direito, clique no ícone de engrenagem e selecione "Install Add-on From File" (ou "Instalar complemento a partir do arquivo").

Navegue até o local onde o arquivo XPI foi baixado e selecione-o.

O Firefox solicitará confirmação para instalar a extensão. Clique em "Add" (ou "Adicionar").

Clique no botão abaixo para instalar a extensão automaticamente.
