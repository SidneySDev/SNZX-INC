Instalar Extensão do Firefox

Instalação Manual no Firefox

Abra o Firefox.

Vá para o menu Extensões e Temas.

No canto superior direito, clique no ícone de engrenagem e selecione "Instalar complemento a partir do arquivo".

Navegue até o local onde o arquivo XPI foi baixado e selecione-o.

O Firefox solicitará confirmação para instalar a extensão. Clique Adicionar.

Clique no botão abaixo para realizar o Download.
